# Snehith Krishnan R — Portfolio Website

A personal portfolio website built with HTML and CSS.

## Pages

- **index.html** — Home page with hero section, bio and skills
- **about.html** — Detailed about page with education, skills and interests
- **projects.html** — Project showcase and CTF activity section
- **contact.html** — Contact info and message form
- **style.css** — All styles in one file

## How to view locally

Just open `index.html` in any browser. No build step needed.

## Stack

- HTML5
- CSS3 (glassmorphism design, CSS variables, responsive layout)
- Font Awesome 6 (icons via CDN)
- Google Fonts — Cormorant Garamond + DM Sans
